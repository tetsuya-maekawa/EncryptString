# EncryptString

## 概要
パスワードを指定して、文字列を暗号化、又は複合します。

## 引数
* 第一引数：処理モード
　/e ... 暗号化、/d ... 複合
* 第二引数：対象の文字列
　/eの場合 ... 平文、/dの場合 ... 暗号化済みテキスト
* 第三引数：パスワード 

## 使用例
暗号化する場合

``` EncryptString.exe /e [暗号化したい文字列] [PASSWORD] ```

複合する場合

``` EncryptString.exe /d [暗号化済文字列] [PASSWORD] ```
